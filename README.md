# shopping_list_app

ระบบแสดงผลและจัดกลุ่ม: ระบบดึงข้อมูลรายการสินค้ามาแสดงผล พร้อมจัดกลุ่มแยกตามหมวดหมู่ให้อัตโนมัติ และแสดงไอคอนประจำหมวดหมู่

ระบบเพิ่มรายการ: หน้าฟอร์มสำหรับกรอกชื่อสินค้า จำนวน และเลือกหมวดหมู่ พร้อมระบบตรวจสอบความถูกต้องของข้อมูล ก่อนบันทึก

ระบบแก้ไขรายการ: ระบบแตะที่รายการ (Tap to Edit) เพื่อดึงข้อมูลเดิมขึ้นมาแก้ไขรายละเอียด และอัปเดตทับข้อมูลเดิม

ระบบลบรายการ: ระบบปัดหน้าจอไปทางซ้าย เพื่อลบรายการที่ต้องการทิ้ง

ระบบจัดการสถานะ: ระบบกล่องทำเครื่องหมาย (Checkbox) เมื่อกดติ๊ก ระบบจะทำการขีดฆ่าข้อความและปรับเป็นสีเทา เพื่อแสดงสถานะว่าหยิบของลงตะกร้าแล้ว

ระบบเชื่อมต่อฐานข้อมูล: ระบบเบื้องหลังที่เชื่อมต่อ API กับ Firebase Realtime Database เพื่อบันทึกและซิงค์ข้อมูลทั้งหมดแบบออนไลน์
